Firebase Database Connection

Sync products and orders with the cloud in real-time. Follow the steps below to initialize your free Google Firebase project.

1

Create a Firebase Project

Go to the Firebase Console, click Add Project, and give it a name (e.g. Pony Flowers).

2

Enable Firestore Database & Authentication

In the console sidebar, go to Build ➔ Firestore Database and click Create database (start in Test mode). Then go to Build ➔ Authentication, click Get Started, and enable the Email/Password provider.

3

Register a Web App

Go to Project Settings (gear icon), click the </> Web App icon, and register your application. Copy the `firebaseConfig` object variables displayed in the snippet.
